# Contexto Română 🇷🇴

Contexto Română este un joc de ghicit cuvinte bazat pe proximitate semantică. Obiectivul tău este să găsești cuvântul secret al zilei. Cu cât cuvântul pe care îl introduci este mai apropiat ca înțeles de cel secret, cu atât rank-ul acestuia va fi mai mic.

![App Screenshot](https://via.placeholder.com/800x400?text=Contexto+Romana+UI)

## ✨ Caracteristici
- **Algoritm Semantic**: Utilizează word embeddings pentru a calcula distanța dintre cuvinte.
- **Interfață Modernă**: Design premium cu mod întunecat și animații subtile.
- **Dicționar Extins**: Peste 50.000 de cuvinte în limba română.
- **Responsive**: Funcționează excelent atât pe desktop, cât și pe mobil.

## 🚀 Instalare Locală
Dacă dorești să rulezi jocul pe propriul calculator, urmează acești pași:

1. Clonează repository-ul:
   ```bash
   git clone https://github.com/LeoM965/Contexto_Romana.git
   cd Contexto_Romana
   ```

2. Instalează dependențele:
   ```bash
   pip install -r requirements.txt
   ```

3. Pornește aplicația:
   ```bash
   streamlit run app.py
   ```

## 🛠 Tehnologii Utilizate
- [Streamlit](https://streamlit.io/) - Framework pentru interfața web.
- [Gensim](https://radimrehurek.com/gensim/) - Biblioteca pentru procesarea limbajului natural (NL).
- [Hugging Face](https://huggingface.co/) - Gazduire pentru modelele de word embeddings.
